# Αναφορά Εντολών TopClaw (Ελληνικά)

Αυτή είναι σελίδα-γέφυρα για γρήγορη πλοήγηση στις εντολές CLI του TopClaw.

Αγγλικό πρωτότυπο:

- [../../commands-reference.md](../../commands-reference.md)

## Πότε να τη χρησιμοποιήσετε

- Για να βρείτε εντολές ανά εργασία (`bootstrap`, `status`, `doctor`, `channel`, `service`)
- Για να ελέγξετε aliases και την τρέχουσα επιφάνεια του CLI
- Για να επιβεβαιώσετε τη συμπεριφορά μετά από πρόσφατες αλλαγές στο αγγλικό εγχειρίδιο

## Κανόνας

- Τα ονόματα εντολών, flags, κλειδιών config και η ακριβής συμπεριφορά runtime ορίζονται στην αγγλική τεκμηρίωση.
- Η πλήρης σύμβαση CLI παραμένει στο αγγλικό πρωτότυπο.

## Σημεία προσοχής

- Το `topclaw bootstrap` είναι η κανονική εντολή αρχικής ρύθμισης για την τρέχουσα έκδοση.
- Για κανάλια που τρέχουν μόνιμα στο παρασκήνιο, ξεκινήστε από `topclaw service status`.
- Μετά το onboarding και αφού ρυθμιστεί channel, το πρώτο λειτουργικό test είναι μήνυμα στο bot του channel.
